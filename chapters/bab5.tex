\chapter{PENUTUP}

\section{KESIMPULAN}

Berdasarkan pelaksanaan kerja praktik yang dilakukan di \NamaPerusahaan\ selama
periode \PeriodeKP, dapat disimpulkan sebagai berikut:

\begin{enumerate}[label=\arabic*., leftmargin=*, itemsep=0pt]
  \item Kesimpulan pertama berkaitan dengan pencapaian tujuan kerja praktik.
  \item Kesimpulan kedua berkaitan dengan hasil proyek yang dikerjakan.
  \item Kesimpulan ketiga berkaitan dengan kompetensi yang diperoleh selama
        pelaksanaan kerja praktik.
\end{enumerate}

\section{SARAN}

Setelah selesai menjalani kerja praktik dan mengerjakan laporan ini, penulis
menyampaikan beberapa saran sebagai bahan masukan bagi berbagai pihak.

\subsection{Saran untuk Perusahaan (\NamaPerusahaan)}

\begin{enumerate}[label=\arabic*., leftmargin=*, itemsep=0pt]
  \item Saran bagi instansi terkait pengembangan sistem atau proses kerja
        di masa mendatang.
  \item Saran kedua untuk instansi.
\end{enumerate}

\subsection{Saran untuk Pihak Kampus (Politeknik Elektronika Negeri Surabaya)}

\begin{enumerate}[label=\arabic*., leftmargin=*, itemsep=0pt]
  \item Saran bagi program studi terkait persiapan mahasiswa sebelum kerja praktik.
  \item Saran kedua untuk program studi.
\end{enumerate}

\subsection{Saran untuk Mahasiswa Selanjutnya}

\begin{enumerate}[label=\arabic*., leftmargin=*, itemsep=0pt]
  \item Saran bagi mahasiswa yang akan melaksanakan kerja praktik di tempat
        yang sama atau bidang yang serupa.
  \item Saran kedua untuk mahasiswa selanjutnya.
\end{enumerate}

% Hapus sub-section berikut jika tidak ada saran pengembangan per proyek.
\subsection{Saran Pengembangan untuk [Nama Proyek 1]}

\begin{enumerate}[label=\arabic*., leftmargin=*, itemsep=0pt]
  \item Saran pengembangan teknis untuk proyek pertama ke depannya.
  \item Saran kedua terkait pengembangan proyek pertama.
\end{enumerate}

\subsection{Saran Pengembangan untuk [Nama Proyek 2]}

\begin{enumerate}[label=\arabic*., leftmargin=*, itemsep=0pt]
  \item Saran pengembangan teknis untuk proyek kedua ke depannya.
  \item Saran kedua terkait pengembangan proyek kedua.
\end{enumerate}
